# PeRank — identità visiva

## Colori
- Navy   `#2B3A67`  — colore primario (cerchio, barre scure, testi forti)
- Corallo`#FF4D6D`  — accento (freccia giù, barre chiare)
- Bianco `#FFFFFF`  — freccia su, controcampo

## File
- `perank-mark.svg`  — marchio (cerchio + frecce). Sorgente delle icone.
- `perank-logo.svg`  — logo completo (barre + marchio). Store, hero page, promo.
- `perank-mark-512.png`, `perank-logo-512.png`, `perank-logo-1024.png`

## Uso
- **Icone estensione** (16/32/48/96/128): sempre il *marchio*, mai il logo completo:
  a quelle dimensioni le barre diventano illeggibili.
- **Store, hero page, materiali promozionali**: logo completo.
- Sfondo trasparente, nessuna ombra, stile flat.
